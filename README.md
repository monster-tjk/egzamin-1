Муаррифии лоиҳаи "Physionic"

Ин лоиҳа як саҳифаи ягонаи интернетӣ (Landing Page) барои беморхона ва маркази тиббӣ мебошад, ки бо истифода аз коди тозаи HTML ва CSS сохта шудааст.

 Қисмҳои асосии сайт (Структура):

1. **Сароғоз (Navbar):** Логотипи сайт, менюи пайвандҳо барои гузаштан ба бахшҳои дигар ва тугмаи "Get Quote Now" барои гирифтани нархнома.
2. **Бахши асосӣ (Hero Section):** Сурати заминавии тиббӣ, матни истиқболӣ ва формаи махсус барои фармоиши навбати қабули духтур (Book Appointment). Дар ин ҷо корбар метавонад ном, email ва шӯъбаро интихоб кунад.
3. **Хизматрасониҳо (Services):** Се корти кабудранг, ки бахшҳои асосии тиббиро нишон медиҳанд: Кӯмаки таъҷилӣ (Emergency Case), Табобати саратон (Cancer Care) ва Саволу ҷавоби тиббӣ (Health Queries).
4. **Саволҳои зуд-зуд додашаванда (FAQ):** Блоки махсус бо сохтори шабакавӣ (Grid) барои ҷойгир кардани саволҳо ва ҷавобҳои муҳим.
5. **Шарҳи беморон (Testimonials):** Фикру мулоҳизаҳои одамон дар бораи беморхона ҳамроҳ бо рейтинги ситораҳо ва суратҳои доиравии беморон.
6. **Нархнома (Pricing):** Се намуди нақшаи хизматрасонӣ: Бепул (Free), Стандарт (Standard) ва Премиум (Premium).
7. **Формаи обуна (Get In Touch):** Бахши торик барои ворид кардани email ва обуна шудан ба хабарҳои нав.
8. **Поёни сайт (Footer):** Инокаҳои шабакаҳои иҷтимоӣ ва пайвандҳои иловагӣ барои маълумоти бештар.

 Технологияҳои истифодашуда:
* **HTML5** — Барои сохтани сохтор ва қисмҳои асосии сайт.
* **CSS3** — Барои ороиш, рангҳо, ҳуруфи Inter (Google Fonts) ва сохтори Flexbox ва Grid.









# Простой дружелюбный сервис

Ин лоиҳа як саҳифаи веби (Landing Page) муосир ва адаптивӣ мебошад, ки хизматрасониҳои сервисро муаррифӣ мекунад. Блокҳои асосии он вобаста ба андозаи экрани корбар (Ноутбук, Планшет ва Телефон) намуди худро автоматӣ тағир медиҳанд.

## 🚀 Хусусиятҳои лоиҳа (Особенности)

* **Работает онлайн:** Веб-сервис работает везде, где есть интернет. Обновляется бесплатно, автоматически.
* **Лёгкий в освоении:** Сервис можно освоить за 2 дня, он простой и понятный. Есть подсказки, напоминания.
* **Ускоряет работу:** Автозаполнение по ИНН, распознавание накладных, загрузка данных из 1С.

## 🛠 Танзимоти техникӣ (Технологии)

* **HTML5** — Барои сохтор ва анатомияи сайт.
* **CSS3 (Flexbox & Media Queries)** — Барои стилҳо ва адаптивност (ҳам барои телефон ва ҳам барои компютер).

## 📱 Адаптивност (Responsive Design)

Лоиҳа комилан адаптивӣ сохта шудааст:
1. **Компютер ва Ноутбук:** Блокҳо ба таври уфуқӣ (горизонталӣ) ҷойгир шудаанд.
2. **Планшет ва Телефон:** Блокҳо ба таври амудӣ (вертикалӣ) зери якдигар меоянд ва матну расмҳо дар марказ (center) ҷойгир мешаванд.
